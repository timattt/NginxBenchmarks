# Nginx benchmarks

Простые тесты с балансировкой и кэшированием для nginx

## Как дебажить

1. Устанавливаем докер
2. Устанавливаем java
3. Переходим в папку ```scripts/debug```
4. Запускаем команду ```docker compose up```
